FIND CNPJ

Find CNPJ é uma aplicação web desenvolvida para simplificar o processo de consulta de informações de empresas no Brasil usando o número do CNPJ. Exibe rapidamente dados importantes como nome da empresa, endereço, situação cadastral e inscrição estadual. Novas funcionalidades estão em desenvolvimento, incluindo leitura de notas fiscais (NF), para tornar a ferramenta ainda mais completa.
